## Dataset and Batches

The dataset and batch files are not uploaded to this repository because they are large in size. Instead, the project works with the local CIFAR-10 dataset files stored in the `data/` directory.

### CIFAR-10 Data Details
- **Dataset name:** CIFAR-10
- **Total images:** 60,000
- **Image size:** 32 × 32 pixels
- **Channels:** 3 (RGB color images)
- **Classes:** 10
- **Split:** 50,000 training images and 10,000 test images

### Batch Files
The dataset is organized into CIFAR-10 batch files:
- `data_batch_1`
- `data_batch_2`
- `data_batch_3`
- `data_batch_4`
- `data_batch_5`
- `test_batch`
- `batches.meta`

These batch files are used for loading and processing the dataset during training and testing. Since they are large, they are intentionally not included in the GitHub repository.
